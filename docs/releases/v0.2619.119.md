## Highlights

- Added MCP introspection commands across HTTP, SSE, and STDIO transports: `tools`, `refresh`, `describe`, and `schema`.
- Added `--json` as a shorter alias for `--json-output`, including late-position target flags.
- Improved STDIO MCP calls by loading the tool schema before parsing arguments, so string-typed values such as `[]` stay strings when the schema requires it.
- Preserved unknown flags passed during `clip add <name> --stdio ...` so server-specific options can be forwarded to the MCP command.

## Developer Notes

- Shared MCP introspection formatting now lives in `extensions/protocol-mcp/src/introspection.ts`.
- Non-CLI targets now bypass ACL checks for MCP introspection commands so discovery commands remain callable even when a target denies tool execution.
- STDIO dry-run uses cached tool schemas when available, making its preview closer to the real call shape.

## Documentation

- Documented `clip <target> tools --json`, `describe`, and `schema` in the MCP docs.
- Updated the README global flag list to include `--json`.

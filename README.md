## Install

```sh
curl -fsSL https://raw.githubusercontent.com/ori-kim/duru/main/install.sh | sh
```

## Install AI Agent Skills

Install duru's AI agent skills (Claude Code, Cursor, Copilot, etc.) via the
[vercel-labs/skills](https://github.com/vercel-labs/skills) ecosystem:

```sh
# install all duru skills
npx skills add ori-kim/duru

# install a specific skill
npx skills add ori-kim/duru <skill>
```

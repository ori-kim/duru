## 설치

```sh
curl -fsSL https://raw.githubusercontent.com/ori-kim/duru/main/install.sh | sh
```

## AI 에이전트 스킬 설치

[vercel-labs/skills](https://github.com/vercel-labs/skills) 생태계를 통해 duru의 AI 에이전트 스킬(Claude Code, Cursor, Copilot 등)을 설치할 수 있어요:

```sh
# 전체 duru 스킬 설치
npx skills add ori-kim/duru

# 특정 스킬만 설치
npx skills add ori-kim/duru <skill>
```
